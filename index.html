  • April 2021

    As I am fond of aviation, the project was created to conveniently store the history of my flights in flight simulators. It has a user-friendly interface with a date-time scale so that you can visually see the duration of any flight. It is able to record the flight through the connection with the flight simulator. You can make a pre-registration of a future flight and see detailed information about it. Automatic estimation of flight distance and conversion of 4-letter airport code to airport city using remote database API. The graphical interface was created using the PyQT5 library, without any additional graphical programs.
